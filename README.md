## 介绍

一个简易的类Notion笔记应用，可以创建你的公开/私人笔记，包含插入图片，回收站等功能。并可发布笔记并预览。

基于 `Next18` 与 `Convex` 开发，并使用`Clerk`作为鉴权服务，Pre-CSS使用`Tailwind`.
